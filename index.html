README

BBC News Labs Bootstrap Theme

This is a BBC News Labs theme for Bootstrap, loosely based on the style of the beta version of the new BBC News responsive site.

This gives us a single theme we can use across BBC News Labs experiments to quickly build prototypes with a unifed apperance.

Created with Themestrap (see themestrap-README.md)
